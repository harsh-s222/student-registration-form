# Student Registration App 📱

This is a simple Android application developed using Kotlin in Android Studio.

## 📌 Features
- Student Registration Form
- Input Fields:
  - Name
  - Roll Number
  - Department
  - Year
  - Email
  - Phone Number
- Input Validation
- Toast Message on Successful Submission

## 🛠 Technologies Used
- Kotlin
- Android Studio
- ConstraintLayout
- XML Layout Design

## 🎯 Learning Outcome
- Understanding Android UI components
- Implementing ConstraintLayout
- Handling button click events
- Applying input validation

## 🚀 How to Run
1. Clone the repository
2. Open in Android Studio
3. Run on Emulator or Physical Device
